# ccm-backend

把大學時期的專案 [China Cipher Messenger](https://github.com/ChinaCipher/messenger-server) 重寫了一次。

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
